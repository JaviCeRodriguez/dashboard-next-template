# Dashboard Next.js Template

Dashboard made with Next.js, ChakraUI & ApexCharts

## How to run

```sh
git clone https://github.com/JaviCeRodriguez/dashboard-next-template.git

cd ./dashboard-next-template

yarn install # Or npm install

yarn dev
```
